Proyecto

Our project aims to raise awareness of the Braille system by explaining how this tactile writing and reading method used by people with visual impairments works. The Braille system uses combinations of six raised dots to represent letters, numbers, and symbols, allowing users to read by touch and write using special tools.

In addition to showcasing information about Braille, our project proposes the development of a physical keyboard based on the Braille system. This keyboard is designed to help people with visual impairments write and communicate more easily on digital devices.

Teclado Fisico

The keyboard would operate using six main buttons, representing the six dots of a Braille cell. By pressing different combinations of these buttons, the user can form letters and words, similar to how the traditional Braille system works.

Objetivo

The objective of this system is to improve technological accessibility, allowing more people to use computers, websites, and applications independently. In this way, we aim to promote digital inclusion and access to information for everyone.
