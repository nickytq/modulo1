Proyecto

Nuestro proyecto tiene como objetivo dar a conocer el sistema Braille, explicando cómo funciona la escritura y la lectura táctil utilizada por las personas con discapacidad visual. El sistema Braille utiliza combinaciones de seis puntos en relieve que representan letras, números y símbolos, permitiendo a los usuarios leer con el tacto y escribir utilizando herramientas especiales.

Además de mostrar información sobre el Braille, nuestro proyecto propone el desarrollo de un teclado físico basado en el sistema Braille. Este teclado está diseñado para ayudar a las personas con discapacidad visual a escribir y comunicarse más fácilmente en dispositivos digitales.

Teclado Fisico

El teclado funcionaría con seis botones principales, que representan los seis puntos de la celda Braille. Al presionar diferentes combinaciones de estos botones, el usuario puede formar letras y palabras, de manera similar a cómo funciona el sistema Braille tradicional.

Objetivo

El objetivo de este sistema es mejorar la accesibilidad tecnológica, permitiendo que más personas puedan usar computadoras, páginas web y aplicaciones de forma independiente. De esta manera, buscamos promover la inclusión digital y el acceso a la información para todos.
